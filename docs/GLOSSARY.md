# GLOSSARY.md — Глоссарий CaucasHub

> Единственный источник терминов. При расхождении кода с глоссарием — обновить код.

---

## Участники

| Термин | Определение |
|--------|-------------|
| **Грузовладелец (shipper)** | Пользователь с ролью `shipper` или `both`. Создаёт грузы, принимает/отклоняет отклики. |
| **Перевозчик (carrier)** | Пользователь с ролью `carrier` или `both`. Откликается на грузы, управляет доставкой. |
| **Администратор** | Доступ через `ADMIN_SECRET`. Управляет спорами, статусами, массовыми операциями. |
| **Система** | Автоматические переходы без участия пользователя (например, при accept отклика). |

---

## Машина (Truck)

| Термин | Определение |
|--------|-------------|
| **Онлайн** | `Truck.is_available = True` И `Truck.created_at >= now() - 24h` (или `updated_at` если добавлено). Отображается в счётчике «N машин онлайн» в шапке. |
| **Доступна** | `Truck.is_available = True` — флаг выставляется перевозчиком вручную. |

> **Код:** `app/routers/stats.py` — cutoff = now - 24h

---

## Груз (Load)

| Статус | Определение |
|--------|-------------|
| `active` | Груз опубликован, принимает отклики. |
| `taken` | Принят отклик, идёт сделка. |
| `expired` | Дата загрузки прошла, груз автоматически деактивирован (cron, не реализован). |
| `canceled` | Грузовладелец отменил груз. Терминальный. |

---

## Отклик (Response)

| Статус | Определение |
|--------|-------------|
| `pending` | Отклик отправлен, ждёт решения грузовладельца. |
| `accepted` | Грузовладелец принял. Создаётся сделка. Терминальный. |
| `rejected` | Грузовладелец отклонил (или авто при принятии другого). Терминальный. |
| `withdrawn` | Перевозчик сам отозвал отклик. Терминальный. |

---

## Сделка (Deal)

| Статус | Определение |
|--------|-------------|
| `confirmed` | Сделка создана, перевозчик должен приступить к загрузке. |
| `loading` | Перевозчик у склада, идёт загрузка. |
| `in_transit` | Груз в пути. |
| `delivered` | Перевозчик/грузовладелец подтвердил доставку (ждём второго). |
| `completed` | Обе стороны подтвердили. Акт выписан. |
| `rated` | Оценка выставлена. Терминальный. |
| `disputed` | Открыт спор. Разрешается администратором. |
| `canceled` | Отменена любой из сторон. Терминальный. |

## Валюта

| Термин | Определение |
|--------|-------------|
| **GEL / ₾** | Грузинский лари. Основная валюта платформы. |
| **USD / $** | Доллар США. Вторая валюта (автоконвертация по курсу NBG). |
| **exchange_rate_snapshot** | Курс GEL/USD из NBG, зафиксированный в момент создания сделки. Не изменяется. |
| **Онлайн** (машина) | Для машин — `is_available=True` И активность за 24ч. |
