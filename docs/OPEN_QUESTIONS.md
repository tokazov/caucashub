## OQ-002: org_type — свободная строка или Enum (1.3)

**Вопрос:** Юр. форма компании сейчас — свободная строка `VARCHAR`. Нужен ли Enum с фиксированным списком `(llc, ie, jsc, private)`?

## OQ-003: Валюта откликов — только GEL или dual-currency (1.6)

**Вопрос:** `Response.price_usd` — единственное поле цены в отклике. При сделке груз в GEL, отклик в USD. Что конвертировать при создании `Deal.agreed_price`?

**Статус:** PROPOSED → требует явного подтверждения.

## OQ-004: Справочник городов — expand scope? (1.9.5)

**Вопрос:** Сейчас есть `GET /api/cities` (справочник Грузии). Но `from_city`/`to_city` принимают любую строку, в том числе международные (Москва, Ереван).

**Варианты:**
- A: Для local-грузов — только из справочника (`select`). Для intl — свободный ввод.
- B: Два отдельных поля: `from_city_id` (FK) для Грузии + `from_city_intl` (VARCHAR) для международных.

**Статус:** OPEN.

## OQ-005: Верификация перевозчиков — процесс (2.4.4)
