## OQ-004: Справочник городов — expand scope? (1.9.5)

**Вопрос:** Сейчас есть `GET /api/cities` (справочник Грузии). Но `from_city`/`to_city` принимают любую строку, в том числе международные (Москва, Ереван).

**Варианты:**
- A: Для local-грузов — только из справочника (`select`). Для intl — свободный ввод.
- B: Два отдельных поля: `from_city_id` (FK) для Грузии + `from_city_intl` (VARCHAR) для международных.

**Статус:** OPEN.

## OQ-007: SMS-сервис для подтверждения смены телефона

**Вопрос:** Сейчас код подтверждения при смене телефона отправляется на email пользователя (компромисс). Оптимально — SMS на новый номер.

**Текущая реализация:** `POST /api/users/me/request-phone-change` → код на email. Работает.

**Что нужно для SMS:** Подключить SMS-провайдер (SMSAERO, Twilio, Magti/Silknet для Грузии). Стоимость: ~$0.05-0.10/SMS.

**Условие:** При достижении 50+ активных пользователей — подключить SMS вместо email.

**Статус:** OPEN — отложено до роста.

## OQ-009: UX-проверка перед запуском рекламы

**Вопрос:** Аудит Категория 6 проверяется через автоматизированные E2E-тесты. UX-сторона (понятность текстов, скорость UI, психологическая лёгкость) не будет проверена живым пользователем до запуска.

**Срочность:** Обязательно ДО первой рекламной кампании.

**Решение:** Тимур или приглашённый перевозчик/грузовладелец проходит сценарии A1–B4 из `UX_SCENARIOS.md` руками минимум за день до старта рекламы.

**Статус:** OPEN — блокер для рекламы.

## OQ-010: Внешний геокодер (Yandex/LocationIQ)

**Вопрос:** Запрос на Yandex Advanced не отправлен. Сейчас работает собственная таблица 45 городов. Для реальной биржи нужен геокодер с покрытием сёл и районов Грузии.

**Варианты:**
- **Yandex Advanced** — 3–7 дней на ключ, платно, хорошее покрытие Грузии/СНГ
- **LocationIQ** — 1 час на ключ, OSM-данные, бесплатно до 5000 запросов/день. Достаточно для старта.

**Срочность:** Обязательно ДО запуска рекламы — пользователи будут вводить не только города из таблицы 45.

**Статус:** OPEN — блокер для рекламы.
