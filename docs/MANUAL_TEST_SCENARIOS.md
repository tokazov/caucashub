## Сценарий A — Полный цикл грузовладельца

### A1: Регистрация и первый вход
1. Открыть caucashub.ge в инкогнито
2. Нажать «Регистрация»
3. Заполнить: email, пароль (мин. 8 символов), компания, телефон, роль «Грузовладелец»
4. ✅ Ожидаем: вход произошёл, аватар появился в шапке

### A2: Размещение груза
1. Нажать «+ Разместить груз» (или «➕ Груз» в нижней навигации)
2. Заполнить: Откуда (Тбилиси), Куда (Батуми), Вес 2000 кг, Цена 500 ₾, Тип кузова Тент
3. Нажать «Разместить»
4. ✅ Ожидаем: груз появился в списке на главной странице

### A3: Просмотр откликов
1. Кабинет → вкладка «📦 Грузы»
2. Найти свой груз
3. ✅ Ожидаем: виден счётчик откликов

### A4: Принятие отклика
1. Нажать «Принять» на одном из откликов
2. ✅ Ожидаем: статус груза изменился, появилась сделка в «🤝 Сделки»

### A5: Управление сделкой
1. Кабинет → «🤝 Сделки»
2. Нажать «Подтвердить получение» когда груз доставлен
3. ✅ Ожидаем: сделка завершена, появилась форма оценки

### A6: Оценка перевозчика
1. Поставить оценку 1-5
2. ✅ Ожидаем: сделка перешла в статус `rated`

## Сценарий B — Полный цикл перевозчика

### B1: Поиск груза
1. Войти как перевозчик
2. Просмотреть список грузов
3. Применить фильтр: Откуда = Тбилиси
4. ✅ Ожидаем: список отфильтровался

### B2: Отклик на груз
1. Нажать «Откликнуться» на любом грузе
2. Указать цену и комментарий
3. ✅ Ожидаем: отклик появился в Кабинет → «🚛 Отклики»

### B3: Подписка на маршрут
1. Кабинет → «🔔 Подписки»
2. Создать подписку: Тбилиси → Батуми
3. ✅ Ожидаем: подписка появилась в списке

### B4: Telegram уведомление
1. Кабинет → ⚙️ Настройки → «📲 Подключить Telegram»
2. Нажать — откроется t.me/caucashub_bot?start=TOKEN
3. Написать /start в боте
4. ✅ Ожидаем: бот ответил, в настройках появилось «✅ Telegram подключён»

## Сценарий C — Мобильная версия (критично)

### C1: Навигация
1. Открыть caucashub.ge на телефоне
2. Проверить нижнюю навигацию: Грузы / Машины / ➕ / Ставки / Кабинет
3. ✅ Все вкладки открываются

### C2: Форма размещения
1. Нажать «➕ Груз»
2. ✅ Форма открывается снизу как bottom sheet

### C3: Настройки аккаунта
1. Кабинет → шестерёнка ⚙️ в шапке
2. ✅ Настройки открываются на весь экран
3. ✅ Крестик закрывает модалку

### C4: PDF акт
1. Кабинет → «🤝 Сделки» → любая завершённая сделка
2. Нажать «📄 Скачать акт»
3. ✅ PDF открывается с читаемым кирилличным текстом

## Сценарий D — Граничные случаи (Категория 7)

### D1: Пустые состояния
- Новый пользователь: в кабинете везде «нет данных» с подсказками
- Нет грузов по фильтру: «Нет грузов по вашему запросу»

### D2: Валидация форм
- Пароль < 8 символов → ошибка
- Вес 0 кг → ошибка  
- Цена 0 → ошибка
- Дата загрузки в прошлом → ошибка

### D3: Дублирование
- Разместить груз дважды быстро → только один груз (idempotency)
- Откликнуться дважды → только один отклик (409 Conflict)
- Создать подписку дважды → 409

### D4: IDOR проверка
- Попробовать удалить чужой груз → 403/404
- Попробовать принять чужой отклик → 403
